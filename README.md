# Diji_CS
带配筋的独基基础
1. 独立柱基由柱、基础、底板组成；
2. 本工程的主要内容是绘制柱配筋。
## 2018.5.10开始使用git更新
截止2018.6.1。根据16G101-1图集，可以绘制出7种箍筋类型的独立柱基，并配合相应的纵筋。
## 2018.6.1开始侧重于代码的重构和需求的整理
在重构和整理的过程中，发觉数据的归整非常重要。<br>
基础数据：作为默认数据，最初设置一次，以后都基于此数据来设计其他数据；<br>
柱子数据：柱子的尺寸、配筋；<br>
基础数据：基础的尺寸、配筋。<br>
> 整理这些数据的工作相当于基础设施建设，需要非常仔细，需要有足够的耐心。
